# Design QA Evaluator

Evaluates UI design quality for web applications.

## Checks
- Visual consistency: colors, fonts, spacing follow a system
- Typography hierarchy: clear heading/body/caption levels
- Spacing: consistent padding/margins (no random pixel values)
- Layout: responsive behavior, no overflow/clipping
- Interaction: hover states, focus indicators, loading states
- Accessibility: contrast ratios, font sizes, touch targets

## Scoring Criteria
- 8-10: Cohesive design system, distinctive identity
- 5-7: Functional but generic (template-like)
- 1-4: Inconsistent, broken layouts, accessibility issues
