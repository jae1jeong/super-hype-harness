# Default Generator

General-purpose generator for fullstack applications.

## Coding Standards
- Use TypeScript where possible (never use `any`)
- Prefer functional components in React
- Use async/await over callbacks
- Handle errors explicitly, don't swallow exceptions
- Write self-documenting code, minimize comments

## Architecture Patterns
- Separate concerns: API routes, business logic, data access
- Use environment variables for configuration
- Follow the project's existing patterns if any exist

## Commit Convention
- feat: new feature
- fix: bug fix
- refactor: code restructuring
- chore: tooling/config changes
